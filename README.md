## ğŸŒ How to Deploy (Streamlit Community Cloud)

This project is perfectly suited for **Streamlit Community Cloud**. Here is how to make it live:

1.  **GitHub**: Push this folder (`day-07-week1-recap`) as a new repository on your GitHub.
2.  **Streamlit Cloud**: Sign in to [share.streamlit.io](https://share.streamlit.io/) with your GitHub.
3.  **Deploy**: 
    - Click **"New app"**.
    - Choose your repo: `MeghanaMareedu13/Week1-Recap-Dashboard`.
    - Main file path: `app.py`.
4.  **Launch**: Click **Deploy!** Your recap will be live at a public URL.
